简介

尽管Bmob已经提供了一套SDK供开发者使用，但有时候开发者可能希望能直接与Bmob后台进行直接交互，以达到某些特别的需求（直接操作_User表、同步网络请求等）。而RESTful API可以使得只要能够发送HTTP请求的设备可以先Bmob进行数据交互。因此，在使用Bmob开发iOS应用过程中，我们也可以使用RESTful来完成交互。
